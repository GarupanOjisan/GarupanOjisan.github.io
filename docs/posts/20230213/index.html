E: List directory /var/lib/apt/lists/partial is missing. - Acquire (13: Permission denied)

2月 13, 2023 1分で読めます
Tech
Linux

問題 見出しへのリンク

次のようなコマンドを実行したところ次のようなエラーが発生した。

$ apt-get update
$ apt-get install -y gettext

Reading package lists... Done
E: List directory /var/lib/apt/lists/partial is missing. - Acquire (13: Permission denied)

/var/lib/apt/listsとは？ 見出しへのリンク

/var/lib/apt/lists はパッケージ情報のキャッシュを格納する場所です。 aptでパッケージをインストール・更新するときに、このディレクトリで該当パッケージの情報を確認し、 パッケージの詳細が見つかれば、リモートリポジトリから実際にパッケージをダウンロードします。

どうすればよいか？ 見出しへのリンク

要するにこのディレクトリに対して読み書きの権限が足りていないのが原因。 sudoを付けて実行すれば成功した。
